# Asha Tapkigen — Static Portfolio

Plain HTML/CSS portfolio, ready to deploy to GitHub Pages. No build step.

## Deploy to GitHub Pages

1. Create a new GitHub repository.
2. Upload the entire contents of this `static-site/` folder to the **root** of the repo (so `index.html` sits at the top level).
3. In the repo: **Settings → Pages → Build and deployment → Source: Deploy from a branch → Branch: `main` / `/ (root)`**.
4. Your site goes live at `https://<your-username>.github.io/<repo-name>/` within a minute or two.

## Files

- `index.html` — Home (hero, process, certification, CTA)
- `about.html` — About me
- `services.html` — Services
- `portfolio.html` — Case study + TikTok results
- `contact.html` — Contact form (opens user's email client)
- `styles.css` — All styling
- `images/` — Photos
- `certificates/` — Certificate PDF

## Edit

Open any `.html` file in a text editor — the markup is plain HTML with classes from `styles.css`.
